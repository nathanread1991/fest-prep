# Static Images

This directory contains static images for the Festival Playlist Generator web interface.

## Required Images

The following images should be added to this directory:

### Streaming Service Icons (32x32 or 64x64 pixels)
- `spotify-icon.png` - Spotify logo
- `youtube-music-icon.png` - YouTube Music logo  
- `apple-music-icon.png` - Apple Music logo
- `youtube-icon.png` - YouTube logo

### PWA Icons
- `icon-192.png` - 192x192 PWA icon
- `icon-512.png` - 512x512 PWA icon
- `favicon.ico` - Browser favicon

### Screenshots (for PWA manifest)
- `screenshot-mobile.png` - Mobile screenshot (390x844)
- `screenshot-desktop.png` - Desktop screenshot (1280x720)

### Notification Icons
- `badge.png` - Notification badge icon
- `view-icon.png` - View action icon
- `dismiss-icon.png` - Dismiss action icon

## Image Guidelines

- Use PNG format for icons with transparency
- Optimize images for web (compress without losing quality)
- Follow platform branding guidelines for service icons
- Ensure icons are accessible with sufficient contrast

## Placeholder Implementation

For development purposes, you can use placeholder images or CSS-based icons until proper branded icons are obtained from the respective services.
